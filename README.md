- **Sensors**: Temperature/Humidity, Gas (smoke), Tilt (earthquake), Light
- **Potentiometer**: LCD voltage adjustment
- **LCD**: 16x2 display
- **Pushbutton**: Mode selector
- **Piezo**: Alarm for alerts
